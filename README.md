# Xenia Espresso Integration

Custom Integration für die Xenia-Espressomaschine.

## Installation

1. Dieses Repository in `custom_components/xenia` kopieren oder als HACS Custom Repository hinzufügen.
2. Home Assistant neu starten.
3. Unter *Einstellungen → Geräte & Dienste → Integration hinzufügen* „Xenia Espresso“ auswählen.
4. IP-Adresse und Poll-Intervalle konfigurieren.
